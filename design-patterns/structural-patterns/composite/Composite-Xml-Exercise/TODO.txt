Exercise: Composite - XML
---------------------------------------------------------------------

Implementieren Sie die fehlenden Sub-Klassen "XmlElement" und "XmlText",
die zusammen mit der Klasse "XmlNode" ein Composite Pattern bilden
sollen.

Als Operation für die Baumstruktur soll die Methode:

	public abstract String toXML();
	
implementiert werden, welche die Baumstruktur der Objekte als XML String
zurück liefert.

Beachten Sie, dass diese Methode in jeder der beiden Sub-Klassen zu 
implementieren ist. 
	 

Überprüfen Sie Ihre Implementierung mit der Klasse "CompositeTest".
